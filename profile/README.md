# Welcome to DIRECTED Project

We aim to reduce vulnerability to extreme weather events and foster disaster-resilient European societies by promoting interoperability of data, models, communication and governance on all levels and between all actors of the disaster risk management and climate adaptation process.

> :eu: **We are EU funded**
>
> This project is an Innovation Action under the Civil Security for Society, Disaster-Resilient Societies programme of the [Horizon Europe funded by the European Union](https://research-and-innovation.ec.europa.eu/funding/funding-opportunities/funding-programmes-and-open-calls/horizon-europe_en).
